My Journey in Technology

My journey in technology started with curiosity about how websites and software work. At the beginning, I had very little knowledge about programming, but I was always interested in learning new digital skills.

Timeline

HTML Journey

HTML is the foundation of web development and the first step in learning how websites are created. While learning HTML, I understood how web pages are structured using elements like headings, paragraphs, images, links, and lists. It helped me organize content in a meaningful way.

CSS Journey

CSS is used to style and design web pages. While learning CSS, I understood how to control colors, fonts, spacing, margins, padding, and layouts to make websites visually appealing and responsive.

JavaScript Journey

JavaScript adds interactivity to websites. I learned programming concepts such as variables, functions, conditions, loops, and events which helped me build dynamic and interactive web pages.

Tailwind Journey

Tailwind CSS is a utility-first framework that allows developers to design responsive layouts quickly using ready-made classes directly inside HTML.

Bootstrap Journey

Bootstrap is a front-end framework that provides pre-designed components like navbars, cards, buttons, and forms, making it easier to build responsive websites.

React Journey

React is a JavaScript library used to build modern user interfaces using reusable components, making applications more scalable and dynamic.
